# Documentation

This folder groups the non-code project artifacts into one place for review and presentation.

## Structure

- `architecture/`
  - Layered architecture overview
  - Presentation, business, and data responsibilities
- `design-artifacts/`
  - Design notes and references
- `testing-artifacts/`
  - Test strategy
  - Test cases
  - Test results
  - Validation notes
- `presentation/`
  - Presentation-facing summary notes

## Code Areas

- Unity client and gameplay logic: `Assets/`
- Backend API: `backend/`
- Project configuration: `ProjectSettings/`, `Packages/`
